服务端

我们的平台与微信和用户之间交互，这里指的服务端是相对于微信而言的。也就是用户通过微信调用我们，目前也就是消息处理。

消息详见 快速开始 即可。

会自动帮你处理好微信的接入验证请求。如果判断为接入请求的话（ get 请求），会自动处理好的。
